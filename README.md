# Apple OS Simulator

以 HTML、CSS 與 JavaScript 製作的互動式 Apple 系統原型。

## 本版可使用的裝置功能

- 相機：即時預覽、前後鏡頭、拍照與照片圖庫
- 天氣：使用定位取得所在地的即時天氣與五日預報
- 地圖：目前位置、地點搜尋與系統地圖導航
- 電話：撥號鍵盤並呼叫裝置的系統撥號介面
- 設定：裝置辨識、權限狀態、深色模式、模擬器亮度及動作感測器

## 使用要求

相機、定位與動作感測器需要 HTTPS。部署到 GitHub Pages 後，請使用
iPhone／iPad Safari 或 Android Chrome 開啟，並在畫面提示時允許權限。

網頁無法直接讀取系統聯絡人、通話紀錄，或開關 Wi-Fi、藍牙。
